# Gestão Biblioteca Java

Sistema de gerenciamento de biblioteca desenvolvido em Java.

O projeto simula empréstimos e devoluções de livros e DVDs, utilizando conceitos de Programação Orientada a Objetos e recursos da linguagem Java.

## Funcionalidades

- Cadastro de livros e DVDs
- Empréstimo de itens
- Devolução de itens
- Controle de disponibilidade
- Validação de idade para empréstimos

## Conceitos utilizados

- Orientação a Objetos
- Herança
- Encapsulamento
- Polimorfismo
- Collections
- Streams API
- Predicate, Function e Consumer

## Tecnologias

- Java
- Eclipse IDE
